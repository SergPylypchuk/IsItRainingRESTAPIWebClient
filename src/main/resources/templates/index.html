Is it raining?

This client created for testing REST API App 'Is it raining?' that collects data from rain-sensors

Manual mode:


1. Push 'New -> Sensor' add unique name and register one or several sensors.


2. Push 'New measurement', enter some data and choose your sensor.


(You can add many measurements for every sensor)


3. Push 'Show -> Measurement' to see all measurements.


4. Push 'Show -> Rainy days' to see quantity of rainy days.


5. Push 'Clear -> All data' to delete all sensors and measurements.

Batch mode:


1. Push 'Batch -> Sensors' and press 'Generate'.


2. Push 'Batch -> Measurements' and press 'Add'.


(Random data will be generated and added for sensors you've created previously)


3. Push 'Show Measurements' to see all measurements.


4. Push 'Show Rainy days' to see quantity of rainy days.


5. Push 'Clear -> All data' to delete all sensors and measurements.
