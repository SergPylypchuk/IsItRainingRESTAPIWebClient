Is it raining?

This client created for testing REST API App 'Is it raining?' that collects data from rain-sensors

Manual mode:

Push 'New -> Sensor' add unique name and register one or several sensors. Push 'New measurement', enter some data and choose your sensor. You can add many measurements for every sensor. Push 'Show -> Measurement' to see all measurements. Push 'Show -> Rainy days' to see quantity of rainy days. Push 'Clear -> All data' to delete all sensors and measurements.

Batch mode:

Push 'New -> 1000 sensors' and press 'Generate'. Push 'New -> 1000 measurements' and press 'Add'. Random data will be generated on server and added for sensors you create previously. Push 'Show Measurement' to see all measurements. Push 'Show Rainy days' to see quantity of rainy days. Push 'Clear -> All data' to delete all sensors and measurements.
